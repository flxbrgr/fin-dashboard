# CHANGELOG

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-02-24

### Added
- Initial project structure with FastAPI and Vanilla JS.
- "Overnight Overreaction" strategy implementation (+/- 5% detection).
- Integrated `yfinance` for free news fetching (News Traffic Light).
- Natural Language Interface (German) powered by Gemini 1.5 Flash.
- Security layer: JWT Authentication and TOTP 2FA.
- Multi-Watchlist support with per-user persistent storage.
- Ticker search functionality (Yahoo Finance & Binance).
- Premium Dark-Mode UI with glassmorphism and TradingView charts.
